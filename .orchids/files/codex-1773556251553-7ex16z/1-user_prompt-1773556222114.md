You are a senior product designer and expert frontend engineer building a production-quality SaaS level application.

Your task is to design and generate a fully functional, modern, and professional Clothing Brand E-Commerce Website.

Important Rule:
Do NOT generate "vibe coded" UI. The interface must follow proper UI/UX principles, clean layout structure, spacing consistency, accessibility, and professional SaaS-grade design.

Project Requirements:

1. Website Type
Build a complete E-commerce website for a Clothing Brand where both Men and Women can browse, select, and order clothes.

2. Core Features
• Product listing page for Men and Women categories
• Product detail page with images, price, size selection (S, M, L, XL)
• Add to Cart functionality
• Shopping Cart page with quantity update and remove option
• Checkout page with customer details
• Order summary before placing order
• Responsive navigation bar
• Search functionality for products
• Category filtering (Men / Women / New Arrivals / Trending)

3. Pages Required
• Home Page
• Men Collection Page
• Women Collection Page
• Product Detail Page
• Cart Page
• Checkout Page
• About Brand Page
• Contact Page

4. UI / UX Design
• Modern, minimal, premium fashion brand style
• Clean typography and balanced spacing
• Elegant color palette (Black, White, Neutral tones)
• Card-based product layout
• High quality product grid system
• Smooth hover effects on product cards
• Professional navbar with logo and menu

5. Animations & Visual Effects
Use smooth and professional animations to make the website more attractive:

• Smooth hover animation on product cards
• Button hover effects
• Fade-in animation when sections appear
• Smooth page transitions
• Image zoom effect on product hover
• Animated Add-to-Cart feedback
• Smooth scrolling experience

6. Homepage Sections
• Hero Section with promotional banner
• Featured Products
• Men Collection Preview
• Women Collection Preview
• Trending Products
• Brand Story Section
• Newsletter Subscription
• Footer with social links

7. Responsive Design
The website must be fully responsive and optimized for:
• Desktop
• Tablet
• Mobile devices

8. Cart & Order Flow
• User can add products to cart
• Update product quantity
• Remove products from cart
• See total price calculation
• Proceed to checkout

9. Performance
• Clean structured code
• Fast loading design
• Optimized images and layout
• Proper component structure

10. Final Result
The website should look like a professional modern fashion e-commerce store similar to top clothing brands.

The UI must feel premium, smooth, and interactive with elegant animations and a high-end shopping experience.
